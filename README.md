# 🚀 TaskFlow AI - Smart Productivity App

A Smart Task Management Web Application with AI-powered productivity insights, analytics dashboard, calendar integration, and export features.

---

## ✨ Features

- 📝 Add / Edit / Delete Tasks
- 📊 Dashboard (Total, Completed, Pending, Overdue)
- 📅 Full Calendar Integration
- 📈 Charts (Task Analytics using Chart.js)
- 🤖 AI Productivity Score & Suggestions
- 🔍 Search & Filter Tasks
- 🔄 Drag & Drop Task Reordering
- 📤 Export to PDF & Excel
- 🌙 Dark Mode Support
- 💾 Local Storage Support

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Chart.js
- FullCalendar.js
- jsPDF
- SheetJS (Excel export)
